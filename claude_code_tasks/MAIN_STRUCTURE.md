## 🎯 СТРАТЕГИЧЕСКИЕ ПРИНЦИПЫ

### 1. SILO-архитектура (DrMax, гл. 3.2.22 + 5.8)
Сайт ранжируется тем лучше, чем больше страниц посвящено тематике. Вес передаётся снизу вверх: статья (НЧ) → подкатегория (СЧ) → хаб SILO (ВЧ) → главная. Статьи-доноры усиливают материнскую страницу через перелинковку.

### 2. Topical authority через topic clusters (тренд 2026)
Google HCU оценивает экспертизу сайта **целиком**. Правило 2026: ≥25-30 качественных взаимосвязанных статей внутри одного кластера = +40-70% видимости за 3-6 мес.

### 3. YMYL + E-E-A-T (критично для нутрициологии)
Тематика = Your Money or Your Life. Требования:
- **Experience** — контент от первого лица, реальные кейсы участниц
- **Expertise** — везде byline «Наталья Томшина, нутрициолог» → `/about`
- **Authoritativeness** — `/about` с сертификатами, отзывы, истории
- **Trustworthiness** — HTTPS ✅, `/legal/*`, контакты, источники (PubMed)

### 4. Кластеризация по морфологии (важное открытие)
Анализ Яндекс-подсказок показал, что **одно и то же намерение** может разделяться на разные кластеры по формам слов. Яндекс не мерджит автоматически «меню для похудения» и «рацион для похудения» — это **разные SERP, разные страницы**.

Правило: под каждую словоформу, которая имеет свой топ выдачи — **отдельная страница**. См. SILO 6.

### 5. Бренд-правило по терминологии (смягчённое)

**Запрещённые слова «кето / LCHF / low carb» РАЗРЕШЕНЫ в:**
- ✅ SILO 3 «Методы и диеты» (перехват трафика «кето-диета»)
- ✅ SILO 6 «Меню и рационы» (перехват «меню кето диеты»)
- ✅ Рецепты `/recipes/` (огромный трафик «кето рецепты»)

**Запрещены (использовать только «метаболическое питание / метод Натальи / умное питание»):**
- ❌ SILO 1 «Похудение»
- ❌ SILO 2 «Гормоны»
- ❌ SILO 4 «Женское здоровье»
- ❌ SILO 5 «БАДы и витамины»
- ❌ SILO 7 «Отзывы и истории»
- ❌ Лендинги (/, /about, /club, /marathon, /free)

**Логика:** в SILO 3/6 и рецептах мы **встречаем человека там, где он ищет** (например «меню на неделю кето»), а в тексте статьи **переводим его на метод Натальи**. В основных экспертных SILO бренд остаётся чистым.

### 6. Принцип продвижения снизу вверх (DrMax, гл. 2.2)
НЧ-статьи пишутся первыми → быстро в топ → накапливают вес → передают подкатегории → подкатегории выталкивают хаб по СЧ → хабы ранжируются по ВЧ через 6-9 мес.

## 🗺 ПОЛНАЯ КАРТА СТРАНИЦ

### Публичный сайт — nata-tomshina.ru

```
/                             ← главная (лендинг)
/about                        ← страница автора + о проекте (E-E-A-T) ⚠️ критично
/results                      ← отзывы и результаты (хаб SILO 7)
/club                         ← продающий лендинг клуба
/marathon                     ← лендинг марафона
/free                         ← лид-магнит
/partner                      ← лендинг для партнёров (индексируется!)
/join                         ← оплата (Disallow)

/legal/privacy                ← политика конфиденциальности
/legal/oferta                 ← публичная оферта
/legal/disclaimer             ← медицинский дисклеймер ⚠️ критично для YMYL

/blog/                        ← каталог + 7 SILO
  └── [silo]/                 ← хаб SILO (pillar 3000-5000 слов)
      └── [subcategory]/      ← подкатегория (хаб кластера)
          └── [slug]/         ← статья (НЧ/СЧ)

/recipes/                     ← каталог рецептов
  └── [category]/             ← SEO-категория
      └── [slug]/             ← карточка (Schema.org Recipe)
```

> ℹ️ **Про страницу автора:** используем ТОЛЬКО `/about`. Отдельный раздел `/authors/[slug]/` не нужен — он имеет смысл только при нескольких экспертах. Если в будущем появятся приглашённые эксперты — создадим.
