# docs/admin.md — Админка Наташи

## Роут: /admin (только Наташа)

## Разделы админки

### 1. Участницы
- Список всех участниц (email, имя, статус, дата входа)
- Поиск по email
- Статус: триал/активна/истекла
- Клик → профиль участницы

### 2. Уроки мини-курса
Форма добавления/редактирования урока:
- Заголовок
- Описание
- Video ID (Kinescope)
- PDF-ссылка (Google Drive)
- Домашнее задание
- Порядковый номер
- Бесплатный (is_free)

### 3. Вебинары
Форма:
- Название + описание
- Обложка (эмодзи или ссылка)
- Ссылка Kinescope
- Категория (10 тем)
- Порядок
- Доступ по статусу: 3 мес / 6 мес / 12 мес
- Цена разовой покупки (₽) — покупка через Telegram @natasha_tomshina

### 4. Рационы PDF
Форма добавления рациона:
- Название
- Описание
- Ссылка на PDF (Google Drive или прямая)
- Тип (2-разовое, 3-разовое, из курицы и т.д.)
- Активен (показывать/скрыть)

### 5. Я и моё тело
Форма добавления материала:
- Формат: статья / видео / аудио
- Название + описание
- Ссылка (Kinescope / Google Drive / текст)
- Категория (10 тем)
- Порядок
- Цвет карточки
- Показывать первым (NEW 14 дней)
- Доступ в триале (не более 3 тем)

### 6. Медитации
Форма:
- Название + описание
- Формат: аудио / видео
- Ссылка
- Категория: Утренние / Вечерние / До еды / Стресс / Сон
- Эмодзи обложка
- Порядок
- Цвет градиента

### 7. Рассылка
- Написать письмо всем / по сегменту / одной участнице
- Сегменты: quick_start / mama / health / maintain
- Фильтры: триал / активные / истекшие
- Отправка через Resend

### 8. Статистика
- Всего участниц
- Активных / триал / истекших
- Новых за неделю/месяц
- Конверсия триал → оплата

## Сегменты участниц (из онбординга)
- `quick_start` — хочет результат к событию, 25-35 лет
- `mama` — после родов, дети дома
- `health` — есть диагнозы, мягкий вход
- `maintain` — уже худела, удержание результата

## Онбординг-квиз (5 шагов)
1. Имя, возраст, вес, рост, цель, город
2. Здоровье: ЖКТ / диабет / щитовидка / холестерин / СПКЯ / почки / ничего + аллергии
3. Питание: как ест, сколько раз, тяга к сладкому, завтракает ли
4. Опыт похудения: что пробовала, почему бросала
5. Главная цель: быстро / постепенно / энергия / здоровье / удержание

## Email серия триала (Make.com + Resend)
- День 0: Welcome + ссылка на ЛК
- День 1: «Как прошёл завтрак?»
- День 2: «Открытие дня»
- День 3: «Что изменилось за 3 дня»
- День 4: Тизер закрытых разделов
- День 5: История участницы похожего сегмента
- День 6: Прямое предложение вступить
- День 7 утро: «Последний день»
- День 7 вечер 20:00: «Через 4 часа доступ закроется»
