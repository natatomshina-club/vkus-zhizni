# Документация проекта (Obsidian Vault)

Эта папка — база знаний проекта. Открой её в [Obsidian](https://obsidian.md) как Vault: File → Open folder as vault → выбери `docs/`.

## Начать с

- `00-INDEX.md` — главная карта документации
- `01-quickstart.md` — быстрый старт, ключевые команды
- `_templates/CLAUDE-RULES.md` — правила работы Claude Code с этой папкой

| Папка | Назначение |
