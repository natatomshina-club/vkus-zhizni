## ⚠️ ПРОБЛЕМЫ — решить в следующей сессии

### 1. Шаблон письма OTP
Письмо приходит с дефолтным английским текстом вместо красивого HTML.
Нужно настроить шаблон в Supabase self-hosted.
HTML шаблон есть в облачном Supabase → Authentication → Email Templates → Magic Link.
Переменная: `GOTRUE_MAILER_TEMPLATES_MAGIC_LINK` в `/opt/supabase/docker/.env`

### 2. Ошибка в чатах
```
Could not find a relationship between 'channel_posts' and 'members' in the schema cache
```
Скорее всего нужно обновить schema cache в Supabase или добавить foreign key.

### 3. Умная кухня не выдаёт рецепты
Просит заполнить профиль, хотя профиль заполнен и КБЖУ посчитаны.
Возможно проблема в том что запросы идут на неправильный Supabase URL или SERVICE_ROLE_KEY не работает.

### 4. CloudPayments
Не проверена — нужно протестировать оплату.
Ключи пустые в `.env.production.local`: `NEXT_PUBLIC_CLOUDPAYMENTS_PUBLIC_ID=`

### 5. Закоммитить изменения в git
Файлы созданные вручную на сервере не в git:
- `/root/app/src/app/api/auth/check-member/route.ts`
- `/root/app/next.config.ts` (добавлен `output: 'standalone'`)
- `/root/app/Dockerfile`
